### 이 앱을 만든 이유

- 어떤 문제/불편함을 해결하려고 했나요?

저처럼 블랙잭이라는 게임을 아예 몰랐던 크루들을 위해 앞으로 진행될 블랙잭 미션을 실제로 게임을 플레이해봄으로써 조금이나마 이해를 돕고자 만들게 되었습니다.

- 이 앱을 언제, 어떻게 사용할 건가요?

블랙잭 미션 시작하기 전에 충분히 사용할 것입니다.

- 베팅 기능
