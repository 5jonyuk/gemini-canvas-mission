### 이 앱을 만든 이유

- 어떤 문제/불편함을 해결하려고 했나요?
  - 스프링부트 강의를 듣거나 자료를 봐도 실제로 핵심 개념을 정확히 이해하고 있는지 스스로 체크하기가 어려운 불편함을 해소하기 위해 만들게 되었습니다.
- 이 앱을 언제, 어떻게 사용할 건가요?
  - 출퇴근 시에 가볍게 풀거나 자료를 공부한 뒤 복습용으로 사용

### 주요 기능

- 난이도 선택 기능
- 오답노트 및 해설 제공
- 학습 히스토리 제공
- 이전 문제로 넘어가는 기능

### AI 기능 (해당되는 경우)

- 어떤 AI 기능을 추가했나요?
  - 실시간 문제 생성 기능 (객관식, 주관식, 서술형)
  - 제출한 문제 채점 및 해설 제공 기능
- AI 기능이 앱에서 어떤 역할을 하나요?
