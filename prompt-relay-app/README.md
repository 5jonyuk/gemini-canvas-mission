### 이 앱을 만든 이유

- **어떤 문제/불편함을 해결하려고 했나요?**
  - 이삭과 피즈는 일정이 생기면 따로 기록하지 않고 머릿속으로만 기억하는 습관이 있었습니다. 그로 인해 중요한 일정을 종종 잊어버리는 문제가 발생하였고 왜 캘린더에 기록하지 않는지 고민해본 결과 가장 큰 이유는 **직접 일정을 입력하는 과정이 번거롭기 때문**이라는 결론이 나왔습니다.
    이러한 불편함을 해결하기 위해 메모장에 적듯이 자연어로 입력하면 AI가 내용을 분석해 자동으로 일정을 추가해주는 방식으로 불편함을 해소하고자 하였습니다.
  - 사용자는 형식에 맞춰 일일이 입력할 필요 없이 문장을 작성해 AI에게 요청만 하면 됩니다.

- **이 앱을 언제, 어떻게 사용할 건가요?**
  - 우테코 미션 제출 일정이나 개인 일정이 생겼을 때 직접 캘린더에 입력하는 대신 AI 비서에게 메시지를 보내 일정을 추가할 것 입니다.

### AI 기능 (해당되는 경우)

- 어떤 AI 기능을 추가했나요?
    - 사용자가 자연어로 요청하면 일정 또는 할 일을 자동으로 추가 / 수정 / 삭제
    - 설정된 규칙과 일치하지 않는 요청은 반영되지 않도록 설계
    - 해당 연도의 공휴일을 AI가 분석하여 달력에 자동 표시
    - 캐싱을 활용해 동일 연도에 대해서는 한 번만 AI 요청
- AI 기능이 앱에서 어떤 역할을 하나요?
    - 하드코딩 없이 공휴일 데이터를 동적으로 생성
    - 사용자가 수동으로 입력해야 할 작업을 대신 수행
    - 일정 관리의 번거로움을 줄이고 생산성을 향상
