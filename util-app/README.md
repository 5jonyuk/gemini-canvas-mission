### 이 앱을 만든 이유

- 어떤 문제/불편함을 해결하려고 했나요?
  - 테스트코드를 작성할 때 테스트메서드 네이밍을 의미있고 가독성있게 작성하기위해 고민하는 시간을 줄이고자 만들게 되었습니다.

- 이 앱을 언제, 어떻게 사용할 건가요?
  - 테스트코드를 작성할 때 테스트할 메서드의 시나리오를 한국어로 요청할 것 입니다.

### 주요 기능

- 테스트 관례를 반영한 이름 추천
- 추천한 이유 제공
- 원클릭 복사 제공

### AI 기능 (해당되는 경우)

- 어떤 AI 기능을 추가했나요?
  - 테스트 코드 관례를 기준으로 한국어 분석
  - 테스트 코드 관례에 맞는 네이밍 추천 기능
  - 추천한 이유 제공
- AI 기능이 앱에서 어떤 역할을 하나요?
  - 사용자가 입력한 한국어 문장에서 Given(주체), When(행위), Then(결과)을 정확히 추출해 의도를 분석한 뒤 단순 사전적 표현이 아닌 스프링부트 생태계에서 통용되는 전문 용어를 조합하여 테스트 네이밍을 제안합니다. 또한 단순히 이름만 제공하는 것이 아니라 좋은 테스트 네이밍을 위한 기준과 작성 팁까지 함께 안내하는 가이드 역할을 수행합니다.
